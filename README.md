# facioquo.com

[FacioQuo.com](https://facioquo.com) is the top-level website for the **FaciōQuō** organization. Some of our microsite projects are hosted as subdomains. Find us [on GitHub](https://github.com/facioquo).

> **FaciōQuō** is a Latin-inspired name meaning “I make—whereby.”
> It reflects what we do: build useful things, and create the means for ideas to become real.

If you need anything or want to join our Slack channel, [contact us](https://bit.ly/3ZT6AMJ) for more information.

Please note, our tools and microsites are in separate code repos. They may have a different license type.
